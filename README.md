# 工作流

#### 介绍

兴趣使然，想开发一套自己认为的工作流

项目使用 `vue3` + `elementui-plus`+ `ts` + `Activit7` + `SpringBoot` ，项目从零搭建 可移植性高，无依赖多余框架
